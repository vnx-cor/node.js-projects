Fake Data Manager

Practicing Fetch API through real interactions

Explore Data

Fake Data Manager is a personal project built to apply and reinforce my understanding of the Fetch API and asynchronous JavaScript.

Instead of static examples, this app provides multiple data categories where users can interact with live endpoints and perform real CRUD operations. It focuses on how data is requested, updated, and managed in a typical web application.
